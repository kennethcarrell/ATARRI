# ATARRI
A TESS Archive RR Lyrae Classifier

###### Example usage:
```python
import tkinter
from ATARRI.RRLClassifier import RRLClassifier
root = tkinter.Tk()
theGui = RRLClassifier(root)
root.mainloop()
```
